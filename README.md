# security_start

Это простая программа для безопасного запуска браузера в изолированной «песочнице». Когда вы запускаете браузер через неё, он полностью отрезан от ваших личных файлов на компьютере, поэтому подхватить вирус или скрытый скрипт из сети становится невозможно. 

Все скачанные картинки и документы не попадают сразу на компьютер, а ловятся во временную скрытую папку (изолятор). Чтобы забрать файл в свои реальные загрузки, нужно будет ввести ПИН-код в окне программы.

## Как установить программу

Откройте терминал, скопируйте и запустите эти команды по очереди:

```bash
git clone https://github.com
cd security_start
chmod +x install.sh
./install.sh
```

*Скрипт установки сам проверит ваш компьютер и докачает нужную для изоляции утилиту `bubblewrap`, если её у вас нет (для этого он может попросить пароль от компьютера).*

## Как запускать

После установки терминал можно закрыть. Программу можно запустить двумя способами:
1. Просто найдите её в меню ваших приложений по названию **Security Start** (там будет иконка со щитом). Консоль при этом открываться не будет.
2. Либо напишите в терминале: `security_start`

## Из чего сделана программа
* **Python 3** и библиотека **Tkinter** — на них написаны само графическое окно и логика кнопок.
* **Bubblewrap (bwrap)** — системная утилита Linux, которая создаёт защитную коробку вокруг браузера.
